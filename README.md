# Face Emotion Detection

A **real-time facial emotion recognition system** using **Convolutional Neural Networks (CNN)** and **OpenCV** in Python.  

Detects **seven emotions**: **Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise** from webcam video.
